# 設計メモ：導電性PLA を要素技術として使う

競技かるた電気判定システム向けの要素技術検討。札サイズで造形した導電性PLA は
抵抗 **10〜30kΩ** 程度になる。これを 3 状態識別（無接触 / 札 / 畳）に活かせるか。

## なぜ相性が良いか

1. 抵抗式 3 状態の設計の肝＝「札の側面導通を**意図的に高抵抗**にする」を、
   **物理パーツとして印刷で作り込める**。10〜30kΩ は中間電圧を作る抵抗としてちょうど扱いやすい。
2. 抵抗式は ADC 読みが **µs オーダー**。v3 で判明した内蔵容量タッチの **~40ms 天井を回避**でき、
   「速度命」の要求と 3 状態識別を同時に満たせる可能性がある。

## 使い方の案

### 案A：3状態分圧の「中間抵抗」そのものにする（素直）

```
3V ─[R_ref 固定]─ P0(ADC) ─[札の導電PLA R_pla]─ 指→体→足首バンド─ GND
V_P0 = 3V × R_touch / (R_ref + R_touch)
```

| 状態 | 経路抵抗 | ADC |
|---|---|---|
| 無接触 | ∞（開放） | VCC 近く（高） |
| 札タッチ | R_pla + 体 | 中間（PLA で持ち上げ） |
| 畳に直接 | 体のみ（低め） | 低 |

PLA の利点は「中間」の位置を印刷で再現性よく固定できること（トレース長 / 断面 / インフィルで設計）。

### 案B：抵抗コード化で札を識別／多状態化（アナログボタンラダーの3D版）

札ごと・領域ごとに異なる抵抗値を印刷 → 1 本の ADC で「どの札 / どこを触ったか」を
電圧バンドで区別。固定基準抵抗との分圧でレシオメトリックに読めば電源変動に強い。

### 案C：札ブリッジ方式（足首バンド不要化・速度面で最強）

人体を主経路から外す。畳側に 2 電極（sense / GND）、札の接地端に導電PLA。

```
畳: [sense電極]──(札裏の導電PLAがブリッジ R_pla)──[GND電極]
```

| 状態 | 物理 | ADC |
|---|---|---|
| 札あり（静止） | PLA がブリッジ | R_pla で決まる中間値（再現性◎） |
| 札が取られた（払い） | ブリッジ開放 | VCC（瞬時・明確） |
| 手で畳直接 | 皮膚がブリッジ | 体抵抗バンド（別値） |

「札は上凸で左右端のみ接地 → 払うと接触が失われる」という観察と直結。
最も時間にシビアな「札が取られた瞬間」を人体・足首バンド抜きで µs 検出できる。案B と併用可。

## 注意点（材料特性）

- **体抵抗との大小関係**：案A は経路に体抵抗（乾燥肌で 100k〜数MΩ）が直列に入り、
  10〜30kΩ の PLA が埋もれて分離しにくい。→ PLA をより高抵抗に印刷して体抵抗と同オーダーにするか、
  案C のように体を主経路から外す。
- **異方性**：積層方向(Z)の抵抗は層間で XY より桁違いに大きい。トレース向きで値が変わる → 設計で固定。
- **接触抵抗・ピエゾ抵抗**：電極界面の抵抗が大きく不安定。押圧で抵抗が変わる（逆に力センサに利用も可）。
- **ドリフト**：温湿度・経時変化。→ 固定基準抵抗とのレシオメトリック測定＋ベースライン追従＋
  しきい値バンド（ヒステリシス）で吸収。

## 次の実験

1. 札サイズの PLA 片の実抵抗を micro:bit ADC で実測（固定基準抵抗との分圧、テスター値と照合）。
   押圧・積層方向で値がどれだけ動くか把握。
2. 3 バンド（無接触 / PLA経由 / 直接、または 札あり/なし/手）が重ならない閾値帯になるか確認。
3. 抵抗式を高速サンプリング（v3 の枠組み流用）し、~40ms 天井が消えることを実測。

## 実測（2026-06-01, テスター）

導電性PLAかるた 1 枚の抵抗：**近傍 2 点 ≈ 3.6kΩ / 最遠 2 点 ≈ 7kΩ**（距離にほぼ比例）。
想定（10〜30kΩ）より低い。含意：

- **案A は不適と確定**：体抵抗（100kΩ〜数MΩ）に対し数kΩの PLA は埋もれ、中間バンドを作れない。
- **案C（札ブリッジ）が本命**：低抵抗はむしろ好都合（RC 小・低ノイズ・高速）。「札↔開放」は一発で取れる。

## 多札スケーリング：識別は「チャンネル走査（MUX）」で（抵抗コードは却下）

将来「1 本の ADC で多数の札を識別」したい。ただし優先事項は **識別速度** と
**軽く触れても反応する鋭敏さ**。この観点で **案B（抵抗ラダーで 1 ピン多重化）は却下**する。

抵抗ラダー多重化が優先事項と衝突する理由：
1. 軽いタッチ＝接触抵抗が大きく不安定 → カード固有の抵抗コードが化けて誤認。
2. 同時・連続タッチで合成抵抗（並列）になり、どの札の値にも一致しない → 多点が曖昧。
3. 多数を狭い電圧帯に詰めるほど整定・平均化が要り、速度が犠牲。

採用：**各札を独立の抵抗接点にし、アナログMUX（例 74HC4067, 16:1）で走査**。
識別を「抵抗値」ではなく「どのチャンネルか」に載せる。

```
各札の電極 ──→ 74HC4067(16入力) ──(コモン)──→ P0(ADC, 1本)
              ↑ 選択線 4本(デジタル)
```

| 優先事項 | 抵抗ラダー(1ピン) | MUX走査 |
|---|---|---|
| 軽タッチ感度 | × コード化け | ◎ ch毎にベースライン＋しきい値 |
| 識別速度 | △ 整定要 | ◎ 1ch〜数十µs、16ch全走査 <1ms |
| 同時・連続タッチ | × 曖昧 | ◎ 各ch独立判定（多点OK） |
| どの札か | 電圧帯で推定 | ◎ ch番号で確定 |
| ADCピン | 1本 | 1本（＋選択線4本） |

## フィラメント選定の結論

識別を抵抗値に載せないので **高抵抗フィラメントは不要**。各 ch は **低抵抗のまま
（現行 3.6〜7kΩ で十分、必要なら 10〜50kΩ まで）** が最適：低R＝高速・低ハム・軽タッチで素直。
**MΩ級は逆効果**（ADC ソース抵抗による精度低下・ハム）。高抵抗素材の出番は
「チップを足さず純パッシブで多重化」という、今回の優先事項に合わない場合のみ。

→ 当面の方針：**単chプロトタイプ（案C・低抵抗PLA）を先に固め、多札化は MUX 走査で拡張**
（速度・軽タッチ感度・多点・足首バンド不要を一度に狙える）。74HC4067 を 1 個入手すれば 16 札検証へ。

## 更新（2026-06-02）：案C 却下 → 「接触瞬間 × 位置」検出へ転換

案C（札ブリッジ＝札の変位を検出）には 2 つの致命的問題があり却下：

1. **お手つき検出が成立しない**：手は電極A–Bを都合よくまたがない。2点ブリッジ前提が崩れる。
2. **判定すべきは「触れた瞬間」**：案Cは「札が動いた＝取られた後」を見る検出で、競技の核心である
   接触のその瞬間を取り逃す。取りもお手つきも本質は「手が表面に触れた瞬間」。

捉えるべきイベントは「手が表面に接触した瞬間」→ 人体を回路に入れる抵抗式の閉路（足首バンド／規約上許容）。
静電容量式はホバー＋~40ms天井で却下済み。

### 札タッチ vs 畳タッチを「電気的に」区別するのは難しい（重要）

経路は `指 →(札?)→ 電極 → 体 → 足首バンド → GND`。体抵抗は 100kΩ〜数MΩ、PLA札は 3.6〜7kΩ。
札を介しても電極に直接でも直列抵抗差は数kΩだけ → 体のMΩに埋もれ **ADC値はほぼ同じ**。
**これが案Aのレベル区別が破綻した理由**でもある。材質（札/畳）を抵抗レベルで見分けるのは不可。

電気的に区別したいなら方法は2つ：
- (1) **空間で分ける**：札ゾーンと畳ゾーンを別電極・絶縁（Scrapboxの PET 絶縁層がこれ）。「どの ch」で区別。
  代償：同じゾーンの「札あり/取られた後」は電気的に同じ → ゲーム状態で補完。
- (2) **札の抵抗を大きくする**：高抵抗フィラメントで札を ≳ 体抵抗（MΩ級）にし、札経由＝高R帯／畳直接＝低R帯
  でレベル分離（＝案Aの復活）。代償：ADC精度低下・ハム・速度低下。

### 採用：材質判別を捨て「位置 × タイミング × 盤面ロジック」へ

電子回路に「札か畳か（材質）」を当てさせるのはやめる。代わりに：

- **物理層**：どの位置（ゾーン ch）が・いつ（µs）触れられたかだけを高速・高感度に検出
- **論理層**：その位置に正しい札があるかを盤面情報（札レイアウト）と照合 → 取り or お手つき

3状態の読み替え：
- 無接触 = どの ch も閾値以下
- 正しい取り = 正解札の位置 ch が発火
- お手つき = 正解札以外の位置 ch（または空きマス）が発火

→ 修正アーキテクチャは案Aの体ループDNAを継ぐが、札/畳の区別を「抵抗レベル」ではなく
**「どの電極(ゾーン)が反応したか＝空間」**に載せ、取り/お手つき判定はソフト層に委ねる。
